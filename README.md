# API de indicadores do IBGE
Esse é um projeto simples em Python, com uso de uma API pública e sem interface gráfica.

## Objetivo
Treinar conceitos básicos de API Rest utilizando a linguagem escolhida.

## O que o script faz
Esse script consome a API pública do IBGE utilizando a biblioteca `requests` em Python para consultar o indicador que mede a **porcentagem de mulheres de 15 anos ou mais de idade economicamente ativas no Brasil**, analisando sua série histórica. Os dados retornados são filtrados para que os valores nulos não apareçam no terminal.

## Notebook de estudo
Além do script `.py`, este repositório também contém um arquivo `.ipynb`, no qual concentrei anotações, testes e aprendizados, registrando o processo de criação do ambiente, o entendimento do funcionamento da API do IBGE e o uso de conceitos básicos de Python e da biblioteca `requests` (como seus principais métodos). Essas anotações têm caráter puramente complementar ao script final.
